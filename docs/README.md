# Documentação do Projeto

Esta pasta concentra os documentos de planejamento, definição e acompanhamento do projeto desenvolvido na disciplina de Introdução a Algoritmos/Programação.

---

## Arquivos

* `proposta.MD`: documento com a proposta inicial do jogo, preenchido pelo grupo na Semana 1.
* `README.md`: descrição da organização da pasta de documentação e orientações gerais.

---

## Sugestões de uso

* Registrar decisões importantes do projeto ao longo do desenvolvimento.
* Documentar mudanças de escopo (alterações na ideia original do jogo).
* Manter histórico de evolução do projeto.
* Anotar melhorias planejadas para versões futuras.
* Centralizar informações de planejamento para facilitar o trabalho em grupo.

---

## Observações

A proposta inicial (`proposta.MD`) é um documento obrigatório da Semana 1 e serve como base para o desenvolvimento do jogo.

O arquivo `README.md` desta pasta serve apenas como guia organizacional da documentação.
